## 复合运算符

| 自加+ +   | `i++; //相当于 i = i + 1; `   |
| --------- | ----------------------------- |
| 自减--    | `i–-; //相当于 i = i – 1; `   |
| 复合加+=  | ` i+=5; //相当于 i = i + 5; ` |
| 复合减-=  | `i-=5; //相当于 i = i – 5;`   |
| 复合乘*=  | `i*=5; //相当于 i = i * 5;`   |
| 复合除/=  | `i/=5; //相当于 i = i / 5;`   |
| 复合与&=  | `i&=5; //相当于 i = i & 5;`   |
| 复合或\|= | `i|=5; //相当于 i = i | 5;`   |
