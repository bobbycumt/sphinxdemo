# 一级

------

## ✅ 一级考试总体特点

- **不考复杂算法**
- **代码量小、重基础语法和理解**
- **选择题侧重：语法、输入输出、表达式、判断、循环**
- **编程题侧重：简单循环 / 条件 / 数位 / 求和统计**

------

## 🟦 **选择题 — 高频知识点总结**

| 知识点                    | 内容说明                                                     | 高频程度           |
| ------------------------- | ------------------------------------------------------------ | ------------------ |
| **变量与命名规则**        | 不能以数字开头；可包含字母、数字、下划线；不能包含空格、中文、符号 | ★★★                |
| **基本输入输出**          | `cin >>`、`cout <<`，格式化输出 `printf()`、`scanf()`        | ★★★                |
| **数据类型**              | `int`, `float`, `char`, `string`，整型赋小数会截断           | ★★☆                |
| **运算符优先级**          | 乘除取模 `* / %` 高于 加减 `+ -`                             | ★★★                |
| **取模运算 %**            | 常用于判断奇偶、整除、取末位                                 | ★★★★（核心）       |
| **逻辑运算符**            | `&&` 与、`                                                   |                    |
| **if 分支结构**           | `if / else if / else` 条件判断                               | ★★★★               |
| **switch 选择结构**       | 常考：忘写 `break` 导致穿透                                  | ★★                 |
| **for 和 while 循环**     | 累加、计数、统计、控制循环次数                               | ★★★★               |
| **break / continue**      | `break`跳出循环，`continue`跳过当次循环                      | ★★★                |
| **自增自减运算**          | `i++` 与 `++i` 的区别                                        | ★★                 |
| **位数统计 / 数位分离**   | `n % 10`、`n / 10` 循环处理数字                              | ★★★★（编程题必考） |
| **判断质数 / 完全平方数** | 简单约数判断、`i*i == N` 检查平方数                          | ★★                 |

------

## 🟩 **编程题 — 高频类型总结**

| 类型                         | 示例功能                                | 常用代码核心                 | 高频程度             |
| ---------------------------- | --------------------------------------- | ---------------------------- | -------------------- |
| **求和 / 累加 / 计数**       | 求 1~N 奇数和、偶数个数                 | `for(i=1;i<=N;i++)`          | ★★★★                 |
| **判断输出型**               | 输入数字 → 输出是否满足条件             | `if (N % 2 == 0)`            | ★★★★                 |
| **循环读入直到某特殊值**     | 输入若干整数直到输入 `0` 或 `-999` 结束 | `while(N != -999)`           | ★★★                  |
| **数位处理 / 逆序输出**      | 输入 123 → 输出 321                     | `while(N){ d=N%10; N/=10; }` | ★★★★（重点大题核心） |
| **统计符合条件的数字**       | 统计能被 3 整除的数数量                 | `if(i%3==0)`                 | ★★★                  |
| **最大值 / 最小值 比较更新** | 读入多个数找最大最小                    | `if(x>max) max=x;`           | ★★★                  |

## ⭐⭐⭐ **特别标记的高频核心模板（必须背会）**

### ① 取数位与去除末位

```cpp
int d = N % 10; // 取个位
N = N / 10;     // 去掉个位
```

### ② 求 1~N 中所有奇数之和

```cpp
sum = 0;
for(int i=1;i<=N;i++){
    if(i%2==1) sum+=i;
}
```

### ③ 判断一个数是否能被 2 和 3 同时整除

```cpp
if(N%2==0 && N%3==0)
```

### ④ 统计输入直到结束标记

```cpp
while(N != -999){
    cin >> N;
}
```

### ⑤ 数字反转（镜像数）

```cpp
rst = 0;
while(N > 0){
    rst = rst * 10 + N % 10;
    N /= 10;
}
cout << rst;
```
