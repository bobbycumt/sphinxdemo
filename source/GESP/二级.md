# 二级

## ✅ **一、选择题高频考点总结（带★为历年必考）**

### 1) **计算机基础常识（非编程）** ★★★★☆

| 知识点               | 正确记忆                           | 典型真题来源 |
| -------------------- | ---------------------------------- | ------------ |
| 计算机五大部件       | 输入、输出、存储器、运算器、控制器 |              |
| 内存 vs 外存         | 内存：RAM（可读写）；ROM 只读      |              |
| 操作系统 / 架构      | 现代计算机采用**冯·诺依曼结构**    |              |
| 网络基础             | TCP 建立连接靠 **三次握手**        |              |
| 大模型/AI 引用热点题 | “大模型”= **大语言模型（LLM）**    |              |

✅ **应试口令**：内存选 **RAM**；架构选 **冯·诺依曼**；TCP 连三次握手。

---

### 2) **变量与表达式** ★★★★☆

| 考点               | 正确规则                          | 易错点                             |
| ------------------ | --------------------------------- | ---------------------------------- |
| 变量命名规则       | **首字符不能为数字；不能含 - 号** | `5star` ❌，`star5` ✅，`ccf-gesp` ❌ |
| 取模运算           | `5 % 2 = 1`，`5 % 3 = 2`          | `&&` 与 `%` 联合考多次             |
| 强制类型转换对负数 | `int(-123.123/10) = -12`          | 导向“向 0 取整”                    |

✅ **应试口令**：

* “有小数强转 int —— **向 0 取整**”
* “模运算永远不产生小数”。

---

### 3) **分支 + 逻辑运算** ★★★★★（核心高频）

| 功能          | 正确写法                    | 错误写法                    |
| ------------- | --------------------------- | --------------------------- |
| 判断区间      | `if (6 <= x && x <= 8)`     | `6 <= x <= 8` ❌             |
| 判断偶数/奇数 | `i % 2 == 0` / `i % 2 != 0` | `if(i % 2)` 代表 **奇数**   |
| 组合逻辑      | `!(i % 3)` 表示能被 3 整除  | `(i % 3)` 表示不能被 3 整除 |

示例：、

✅ **应试口令**：
`!(x % k)` → “能整除”。
`(x % k)` → “不能整除”。

---

### 4) **循环执行逻辑：break / continue** ★★★★★（必考）

| 语句       | 执行效果                             | 高频误解             |
| ---------- | ------------------------------------ | -------------------- |
| `continue` | 跳过本次循环**后续语句**，进入下一轮 | 不是结束循环         |
| `break`    | 直接**结束当前循环**                 | 不是全部结束所有循环 |

✅ **应试口令**：
`continue` 不退出，只跳过。
`break` 退出当前层。

---

### 5) **数位处理（核心计算类题）** ★★★★★（必考 + 编程题必用）

| 目的              | 代码                            |
| ----------------- | ------------------------------- |
| 取个位            | `x % 10`                        |
| 去末位            | `x /= 10`                       |
| 求位数            | `while(x){ cnt++; x/=10; }`     |
| 取第 M 位（从右） | `N / div % 10` (`div=10^(M-1)`) |

✅ **应试口令**：
**取末位 `%10`，去末位 `/=10`，取第M位 `/div%10`**

## ✅ **二、编程题高频考点总结**

### ① **逐位取数 / 回文数 / 特殊数类** ★★★★★

| 题目类型               | 思路             | 核心模板代码                               |
| ---------------------- | ---------------- | ------------------------------------------ |
| 回文数（正序==逆序）   | 逆序构造比较     | `rev = rev*10 + n%10; n/=10;`              |
| 含某位 / 位数统计      | 扫描数位         | `while(n){if(n%10==k)++cnt;n/=10;}`        |
| 自守数（平方末尾对齐） | 同时 `/=10` 对比 | `while(N1){ if(N1%10!=M1%10) break; ... }` |

---

### ② **矩阵 / 图形输出类** ★★★★☆

| 图形             | 关键技巧                                     |
| ---------------- | -------------------------------------------- |
| 直角三角结构     | 两层 for：外层行，内层列                     |
| 数字与字母循环   | 用 `%9` 或 `%26` 控制循环                    |
| 栅栏、菱形、斜线 | 用关系式 `i+j==n-1`、`abs(i-mid)+abs(j-mid)` |

---

### ③ **数学枚举 / 计数类** ★★★★☆

| 题型               | 判定条件                | 真题例子 |
| ------------------ | ----------------------- | -------- |

| 幂和数             | 枚举 `2^i + 2^j` 入区间 |          |
| 含数字 3 的数量    | 用 while 逐位扫描       |          |

## ✅ **四、质数与素数判定（二级编程题常选考点）**

| 难度 | 洛谷题号  | 题目名称   | 说明                |
| ---- | --------- | ---------- | ------------------- |
| 简单 | **P5731** | 质数判断   | for(i=2;i*i≤n) 版本 |
| 中档 | **P1217** | 超级回文数 | 含回文+质数         |
| 提升 | **P3799** | 妙妙数     | 需要组合逻辑        |

> 标准板子（必须背）：

```
bool prime(int n){
    if(n<2) return false;
    for(int i=2;i*i<=n;i++)
        if(n%i==0) return false;
    return true;
}
```
