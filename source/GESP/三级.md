## 🌟 高频知识点重点解释（考试必背）

| 知识点               | 必须记住的关键规则                            |
| -------------------- | --------------------------------------------- |
| **补码求真值**       | 若符号为 1 → 先取反 +1 → 加符号负号           |
| **右移 >>**          | 负数右移为 **算术右移** → 补符号位            |
| **奇偶性判断**       | `(x & 1) == 0` → 偶数； `(x & 1) == 1` → 奇数 |
| **a ^= b 交换变量**  | 三次异或可交换两个数，无需临时变量            |
| **string vs char[]** | char[] 结尾自带 `'\0'`，string 长度可变       |
| **循环次数**         | while 中 continue 会导致代码段被跳过          |

| **③ 数组遍历 & 最大/最小值查找** | 遍历 + if 判断               | ⭐⭐⭐⭐   | 找最大/找最符合条件的数      |

## 🎯 高频编程模板（务必掌握）

### **数位拆分（出现次数最多）**

```cpp
int x;
while(x > 0){
    int d = x % 10;
    x /= 10;
}
```

### **统计符合条件数字**

```cpp
if(a % 9 == 0 && a % 8 != 0) cnt++;
```

### **交换两数（必背）**

```cpp
a ^= b;
b ^= a;
a ^= b;
```

### **找最大值（标准格式）**

```cpp
int maxv = arr[0];
for(int i = 1; i < n; i++)
    if(arr[i] > maxv) maxv = arr[i];
```
