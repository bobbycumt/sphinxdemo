## 实操接线

### 常用元件

| 功能                | 类型     | 引脚               |
| ------------------- | -------- | ------------------ |
| led亮灭、闪烁       | 数字输出 | 任意               |
| led亮度调节、呼吸灯 | 模拟输出 | 3、5、6、9、10、11 |
| 按键                | 数字输入 | 任意               |
| 电位器              | 模拟输入 | A0~A5              |
| 光敏传感器          | 模拟输入 | A0~A5              |

### 电机

| 雅颂庭蓝色套件 | 方向 | PWM   |
| -------------- | ---- | ----- |
| MA （A电机）   | 12   | 5     |
| MB （B电机）   | 7    | 6     |
| 正向全速       | 0    | 255   |
| 反向全速       | 1    | 0     |
| 正向调速       | 0    | 0-255 |
| 反向调速       | 1    | 255-0 |
| 停止           | 0    | 0     |
